# CAR RACING ENVIRONMENT SOLVED WITH PPO

## In this project, you will see the code needed for training and testing using PPO deep learning

github: https://github.com/aritzrss/Gym-Car-Racing

- carracing.py to test the model and/or retrain if needed.

- models/ folder to access all of the model during training and /best_model to access the best model generated (first generation)
- models_maxperf_optimized/ to access all of the model during training and /best_model to access the best model generated (second generation)
- logs/ to access hyperparameter values during training
- logs_maxperf_optimized/ to access optuna hyperparameter logs
- videos/ to view some of the test trials
- v2 and train2 are some test files, not currently used.
